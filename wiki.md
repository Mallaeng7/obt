# 📘 RustPlusBot 공식 사용 가이드 (Wiki)

RustPlusBot은 인게임 팀 채팅, 디스코드, 그리고 전용 웹 대시보드를 통해 Rust 서버와 실시간으로 상호작용하는 올인원 스마트 어시스턴트입니다.

본 문서에서는 봇이 제공하는 모든 명령어와 핵심 기능의 사용 방법을 안내합니다.

---

## 💬 1. 인게임 팀 채팅 명령어 (In-game Commands)
게임 내 **팀 채팅창**에서 `!` 기호를 붙여 봇에게 명령을 내릴 수 있습니다.

### 📊 서버 및 정보 조회
- `!pop` : 현재 서버의 접속 인원 및 대기열을 확인합니다.
- `!night` : 다음 밤(또는 낮)이 되기까지 현실 시간으로 몇 분이 남았는지 계산합니다.
- `!events` : 헬기, 화물선, 오일 리그, 잠긴 상자 등 서버 주요 이벤트가 마지막으로 발생한 시간을 조회합니다.
- `!wipe` : 서버의 다음 와이프(초기화)까지 남은 시간을 확인합니다. (플러그인 기능)

### 🕹️ 스마트 기기 제어 및 TC
- `!status [기기명]` : 스마트 스위치/알람의 켜짐/꺼짐 상태와, 도구함(TC)의 유지비가 몇 시간 남았는지 확인합니다.
- `!on [기기명]` / `!off [기기명]` : 스마트 스위치를 원격으로 켜고 끕니다. (예: `!on 자동문`, `!off 히터`)
- `!timer [기기명] [초]` : 스위치를 켠 뒤 지정한 시간(초) 후에 자동으로 끄는 타이머 기능입니다. (플러그인 기능)

### 🛒 자판기 및 거래
- `!vend [아이템명]` : 전체 맵의 자판기 데이터를 스캔하여, 해당 아이템을 판매하는 가장 가성비 좋은 곳의 좌표와 가격을 알려줍니다.
- `!watchfor [아이템명]` : 누군가 자판기에 해당 아이템을 매물로 올리면 즉시 알림을 보내줍니다. (예: `!watchfor 로켓`)

### ⚔️ 전투 및 생존 계산
- `!craft [수량] [아이템]` : 특정 아이템을 제작하는 데 필요한 재료와 소요 시간을 계산합니다.
- `!recycle [수량] [아이템]` : 아이템을 갈았을 때(재활용) 나오는 자원의 총량을 계산합니다.
- `!durability [건축물]` : 문이나 벽을 파괴하는 데 필요한 C4, 로켓 등 폭발물의 최소 필요량과 가성비 조합을 알려줍니다.
- `!decay [현재HP] [건축물]` : 건물이 완전히 부패하여 무너지기까지 남은 시간을 계산합니다.
- `!turret-add [층수]` / `!turret-check` : 반경 40m 이내의 터렛 개수를 계산하여 터렛 간섭 페널티를 방지합니다.

### 💀 팀원 관리 및 통계
- `!death [팀원이름]` : 해당 팀원이 가장 최근에 사망한 좌표와 시간을 알려줍니다. `!death2`를 치면 두 번째로 최근에 죽은 기록을 보여줍니다.
- `!deathpin [팀원이름]` : 사망한 위치를 인게임 맵 마커(핀)로 찍어줍니다.
- `!teamstats` / `!playtime-all` / `!afktime-all` : 팀원들의 총 접속 시간과 잠수(AFK) 시간을 순위별로 보여줍니다.
- `!promote [팀원이름]` : 봇이 팀장 권한을 가지고 있을 때, 다른 유저에게 팀장 권한을 위임합니다.
- `!hidden` : UI의 한계로 보이지 않는 8인 초과 팀의 숨겨진 멤버 리스트를 출력합니다.
- `!remind [시간] [메시지]` : 지정한 시간 뒤에 봇이 팀 채팅으로 알림을 보내줍니다. (예: `!remind 15m 화물선 상자 언락!`)

## 🎮 2. 디스코드 연동 명령어 (Discord Slash Commands)
디스코드 서버의 텍스트 채널에서 `/` 기호를 입력하여 봇을 제어할 수 있습니다.

- `/credentials add` : 봇을 본인의 Rust 계정과 연결하기 위해 FCM 토큰을 주입하는 명령어입니다. 봇 최초 설정 시 단 한 번만 사용합니다.
- `/pair [서버ID] [기기명]` : 인게임의 스마트 스위치나 알람을 봇과 페어링합니다.
- `/device [on/off/toggle] [기기명]` : 디스코드에서 바로 스위치를 켜고 끌 수 있습니다.
- `/status [기기명]` : 디스코드에서 기기 상태 및 TC를 확인합니다.
- `/vend [아이템]` : 인게임의 `!vend`와 동일하며, 디스코드 UI(임베드)로 깔끔하게 보여줍니다.
- `/events` : 최근 발생한 헬기, 카고 등의 이벤트를 조회합니다.
- `/server info` : 현재 연결된 서버의 맵 이미지, 시드, 맵 사이즈 등 전체 요약 정보를 출력합니다.

## 📺 3. 디스코드 자동 트래커 패널
디스코드의 지정된 채널에 **실시간으로 스스로 업데이트되는 현황판(대시보드)**을 띄워줍니다.

1. **서버 정보 패널 (60초 주기)**: 서버 인원, 대기열, 낮/밤 시간 실시간 갱신
2. **이벤트 타이머 패널 (30초 주기)**: 카고, 헬기, 오일 리그, 잠긴 상자가 뜬 지 몇 분이 지났는지 표시
3. **팀원 상태 패널 (30초 주기)**: 현재 팀원들의 접속 여부(🟢/🔴)와 잠수 시간
4. **스마트 기기 패널 (15초 주기)**: 스마트 스위치의 켜짐/꺼짐 상태와 즉시 조작 가능한 디스코드 버튼 활성화
5. **TC (도구함) 패널 (60초 주기)**: 연결된 모든 TC의 남은 자원과 베이스 유지비(Upkeep) 잔여 시간을 프로그레스 바 형태로 표시

## 🌐 4. 웹 대시보드 (Web UI)
브라우저를 통해 봇 웹 대시보드에 접속하면 텍스트가 아닌 **시각적인 UI**로 서버를 완벽하게 통제할 수 있습니다.

- **인터랙티브 맵 (Map)** : 게임 내 지도와 똑같은 화면에서 실시간 팀원 위치, 자판기 위치, 헬기/카고 경로 히트맵 등을 레이어별로 끄고 켜며 볼 수 있습니다.
- **카메라 스테이션 (CCTV)** : 게임 내에 설치한 CCTV 카메라나 드론 카메라의 화면을 웹 브라우저에서 실시간 영상으로 감상하고, 키보드(WASD)를 통해 카메라를 돌릴 수 있습니다.
- **이벤트 타임라인 (Events)** : 누가 언제 접속했고, 카고가 언제 떴는지 타임라인 형태로 시각화하여 한눈에 파악합니다.
- **플러그인 스튜디오 (Plugins)** : 자바스크립트를 조금 다룰 줄 안다면, 브라우저에 내장된 코드 에디터에서 봇의 새로운 기능(명령어)을 직접 코딩하고 즉시 적용할 수 있습니다. 서버를 재부팅할 필요가 없습니다.
